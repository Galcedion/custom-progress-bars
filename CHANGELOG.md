# Changelog

Version changelog for CustomProgressBars.

## 1.2.0 - 26.06.2026

## Quality of Life

- Improving settings page structure and adding name fields for progress bars.
- Improved frontend generation to avoid errors regarding new settings that initially have no value.
- Unified display across bars, making inner-text center for all bars

## Added Features

- Setting for 3D-Effect on the individual progress bar.
- Setting for separation of the progress bars.
- Adding a new gradient version that calculates the actual progress.
- Setting for segmentation of the individual progress bar.

## Fixes

- Better visibility for settings that are depending on another.
- Minor bugfixes.

## 1.1.0 - 14.09.2025

## Quality of Life

- Restructured the settings page for a better overview.

## Added Features

- Setting for the height of the individual progress bar.
- CSS map to assist custom styling.

## Fixes

- Minor bugfixes.

## 1.0.1 - 06.06.2025

### Fixes

- Various bugfixes.
